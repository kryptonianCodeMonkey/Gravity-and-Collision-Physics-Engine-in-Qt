==========================================
INSTALLATION INSTRUCTIONS FOR QT FRAMEWORK
==========================================
1. Go to https://www.qt.io/download
2. Click "Go Open Source" under the (L)GPL v3 license download
3. Scroll down and click "Download"
4. Open your downloaded file (mine was "qt-unified-windows-x86-3.1.0-online.exe")
5. Click Next
6. Click Skip (no need to have a QT account)
7. Click Next
8. Set install folder or keep default, then click Next
9. In the menu on the "Select Components" page, choose what components you would
like downloaded. For my project, you should download the MSVC 2017 and MinGW 7.3.0
files appropriate to your OS (32-bit or 64-bit, depending), Sources, Qt Charts,
Qt Data Visualization and (at the bottom) Developer and Designer Tools. You can
download more if you'd like.
10. Click Next
11. Accept the terms and click Next
12. Click Next
13. Click Install

==========================================
INSTALLATION INSTRUCTIONS FOR VS QT PLUGIN
==========================================
1. Open Visual Studios
2. Click "Tools" on the toolbar
3. Choose "Extensions and updates..." from the drop down menu
4. Type "QT" into the search bar an press enter
5. Click the Qt Visual Studio Tools extension and click install
